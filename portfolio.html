CineFile: Movie DataBase

A browser-based application for searching for and saving movies the user wishes to watch. Application provides multiple ways to search for movies, displays relevant information as well as trailer of choosen movies and allows users to save any movie to a 'Watch List'. Application built as a group project using jQuery and 2 external APIs. Click to visit the GitHub repository or the launched page.
